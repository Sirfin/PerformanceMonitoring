{% extends "base_generic.html" %} {% block content %} Experiment

To get a json representation of a experiment call: /monitor/api/experiments/experiment_name with the corresponding experiment name

{
"experiment_name": "mbe1093",
"jobs": "http://127.0.0.1:8000/monitor/api/experiments/mbe1093/jobs",
"model": "icon"
}
Attribute Explanation
experiment_name The name of the experiment
jobs The link to the list of the jobs that belong to the experiment
model The name of the model this experiment is part of

It is also possible to get a list of all available experiments: /monitor/api/experiments
