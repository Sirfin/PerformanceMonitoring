To get a json representation of a timer call: /monitor/api/timer/id with the corresponding timer id

{
"id": 219,
"timings": "http://127.0.0.1:8000/monitor/api/timer/219/timings",
"timer_name": "total",
"model": "icon"
},
Attribute Explanation
id The unique id of the timer
timings The link to the list of the timings that belong to the timer
timer_name The name of the timer
model The model this timer belongs to

It is also possible to get a list of all available timers: /monitor/api/timer
