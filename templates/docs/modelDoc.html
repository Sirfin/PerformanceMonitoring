To get a json representation of a model call: /monitor/api/models/model_name with the corresponding model name

{
"model_name": "icon",
"experiments": "http://127.0.0.1:8000/monitor/api/models/icon/experiments",
"timers": "http://127.0.0.1:8000/monitor/api/models/icon/timers"
}
Attribute Explanation
model_name The name of the model
experiments The link to the list of the experiments that belong to the model
timers The link to the list of the timers that belong to the model

It is also possible to get a list of all available models: /monitor/api/models
